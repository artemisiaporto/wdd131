Website Content

Site Name

Mixed Elements

Site Purpose

To correlate ground and fire elements and grass and water elements and show how we can see this happening on Pokémons, describing their types and behavior.

Scenarios

Which could be stunning places that we can see elements interacting in and important combination?

Which Pokémon we can connect/remember when we think about mixed elements in nature?

Color Schema

headings: text in black with a background in a gradient from orange to green

paragraphs: text in black, background in white and border in orange or green

background colors: white, orange and green
