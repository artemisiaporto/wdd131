Site Plan | Website Content

Site Name

Mixed Elements

This name represents the elements relation I will present in the website, as ground mixed with fire and grass mixed with water

---------------------------------------------

Site Purpose

To correlate ground and fire elements and grass and water elements and show how we can see this happening on Pokémons, describing their types and behavior.

---------------------------------------------

Scenarios

Question: Which could be stunning places that we can see elements interacting in an important combination?

Answer: Subterraneous places where ground is melted and forests where plants has a important role in the water cicle

Question: Which Pokémon we can connect/remember when we think about mixed elements in nature?

Answer: About ground and fire, we can think of Camerupt, Ground Primal, and Heatran. About grass and water, we can think of Ludicolo, Lotad and Lombre

---------------------------------------------
